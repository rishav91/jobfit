# Mode: Draft Cover Letter

Generate a cover letter only after asking the user and receiving a yes.

## Rules

- One page max.
- Specific to the company and role.
- Use proof from the resume and match report.
- Mention gaps only when useful and framed honestly.
- Avoid generic phrases like "passionate about" and "perfect fit".
- Do not invent company research; cite only what was extracted or verified.

## Outputs

Write to `drafts/` in the run folder:
- `cover-letter.md`
- `cover-letter.docx`
